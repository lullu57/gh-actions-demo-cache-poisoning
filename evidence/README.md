# Evidence

Run logs, screenshots, and marker files from real attack runs go here.

If you successfully run the attack end-to-end on GitHub, drop:
- a screenshot of the malicious PR's bundle-size workflow succeeding
- a screenshot of the release workflow run that detonated
- the contents of `/tmp/cache-poisoning-marker-*.json` (sanitised)

into this directory so future demoers have proof-of-life.

Keep this directory empty in the published repo — its existence is the placeholder. Real evidence should never include actual tokens, real OIDC URLs, or anything that could be replayed.
